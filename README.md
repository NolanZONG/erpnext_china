# ERPNext 中国本地化

## 功能说明
- 界面中文汉化: 在系统自带简体中文的基础上进行进一步人工本地化
- 中国科目表: 系统首次初始化及初始化后创建新公司代码时可选中国会计科目表
- 创建公司时设置默认值
  - **税务设置**
    - 创建税种：
      - 内销增值税
      - 外销增值税
      - 小规模纳税人
      - 一般纳税人
    - 创建税率模板：
      - 13%
      - 0%
      - 1%
  - **公司设置**
    - 分配默认科目
    - 设置仓库库存科目
    - 配置物料组费用科目
    - 设置小数精度尾差科目
- 新增**中国财务报表**工作区
  - **中国资产负债表**
    - 需设置报表项与科目号对照及计算公式
    - 自动检查科目遗漏
  - **中国利润表**
    - 需设置报表项与科目号对照及计算公式
    - 自动检查科目遗漏
  - **中国现金流量表（直接法）**
    - 需预先设置现金流编码
- 初始化配置
  - 基础设置
    - 默认语言：中文
    - 默认地区：中国
    - 隐藏印度专用字段（如PAN）
  - app安装后设置
    - 修改一词多义字段标签（如采购、销售税费明细中的税率）
    - 隐藏本地化不适用字段（如PAN号等印度专用字段）
    - 修改默认系统流水码前缀-改短

## 安装
没有设置默认站点的，`install-app` 时加`--site`参数

```
bench get-app xinyu_erpnext_china https://github.com/NolanZONG/xinyu_erpnext_china.git
bench install-app xinyu_erpnext_china
```
